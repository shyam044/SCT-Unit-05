1)Functions in Python

  • A function is a block of code used to perform a specific task
  • It is defined with the "def" keyword
  • Syntax: 
    def fun():
    ...
  • Types of Functions:
    • a)Built-in Functions:
      • These are the pre-defined functions provided by Python
      • Example:
        • print()
        • len()
    • b)User-defined Functions:
      • These are the functions created by the programmer
      • They are defined with the "def" keyword
      • Syntax: 
        def fun():
    ...
    • c)Lambda function:
      • Function without a name
      • They are defined with the "lambda" keyword
      • Syntax: 
        a = lambda b: b*b
    • d)Recursive Function:
      • A Function that calls itself is called as a Recursive Function
      • Syntax: 
        def fun(n):
    fun(n-1)
    ...
    • e)Methods:
      • Function defined inside the class is called as Methods
      • Syntax: 
        class A:
    def fun(self):
        ...

4)Web Requests

  • In Python, Web Requests are performed by using request methods
  • Types of Web Requests:
  • i)GET Requests:
    • Retrieves data from a server
    • Fetching data
    • Data is sent in the URL
    • Less secure
    • Do not change the data
    • syntax: 
      requests.get("URL")
    • Program
      • Program
      import requests
print(requests.get("https://www.google.com").json())

# output:
# {'obj1':1, 'obj2':2, 'obj':3, ....}
  • ii)POST Requests:
    • Send data to a server
    • Sending data
    • Data is sent in the request body
    • More secure
    • It changes the data
    • syntax: 
      requests.post("URL")
    • Program
      • Program
      import requests
print(requests.post("https://www.google.com").json())

# output:
# {'obj1':1, 'obj2':2, 'obj':3, ....}

5)Input validation in Python

  • i)check the input: 
    a = input()
if a.isalpha():
    print("Valid")
else:
    print("Invalid")
  • ii)check the length of the input: 
    a = input()
if 5 < len(a) < 10:
    print("Valid")
else:
    print("Invalid")
  • iii)Sanitize the input: 
    a = input()
print(a.replace("<", "").replace(">", ""))
  • iv)Don't use eval() or exec(): 
    # avoid
eval(input())
exec(input())

7)Looping Statements

  • Python mainly provides two looping statements:
    • a)for loop
    • b)while loop
  • a)for loop:
    • It executes a block of code in sequence
    • syntax: 
      for i in range(n):
    ...
  • b)while loop:
    • It executes a block of code until the condition is false
    • syntax: 
      while(condition):
    ...
